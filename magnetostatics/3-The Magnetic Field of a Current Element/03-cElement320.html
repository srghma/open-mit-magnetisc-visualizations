SUBJECT: The Magnetic Field of a Current Element

DESCRIPTION: A simulation of the magnetic field generated by a current element. The observation point (black sphere) can be moved using the arrow keys to sample the field at different positions relative to the current element. The red arrow at the current element and the transparent red arrow at the observation point indicate the direction of current flow. The transparent orange arrow at the observation point indicates the direction from the current element to the observation point. The blue arrow represents the magnetic field at the observation point.
